# ciscos

A flutter firebase ecommerce project built during the internship.
 
Screens it contains:
=> Onboarding

=> Login (Google and email)

=> Forgot Password

=> Sign Up

=> Complete Profile

=> OTP Verification

=> Home Page

=> Product Details

=> Order

=> Profile 

=>Settings

=> Testimonials

=> Suggestions and expert opinion 

=> Bottom Navigation Bar

=> Cart

=> Checkout and Payment
